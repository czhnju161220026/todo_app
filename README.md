# React(antd) + Django实现一个待办清单

## 进度

+ 可以增、删、标记一个待办事项为完成
+ Todo：编辑一个现有的待办事项



## 运行方法

+ 后端：
  + 安装django ，djangorestframework， django-cors-headers
  + python manage.py runserver 8000
+ 前端：
  + 安装cnpm
  + cnpm install
  + cnpm run dev
